## 🧠 What I Learned
 
Working on this project helped me practice:
 
- **CSS Grid named areas** — using `grid-template-areas` to build a complex bento-style layout without relying on hardcoded positions
- **Flexbox inside grid items** — combining grid for the macro layout and flexbox for content alignment within each card
- **Responsive design** — reorganizing an entire grid into a single-column layout using one media query
- **CSS custom properties** — keeping colors and spacing consistent and easy to update across the stylesheet
- **Code organization** — structuring CSS with section comments and BEM-style class naming (`card--header`, `card--platforms`) for readability
